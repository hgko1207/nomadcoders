## React Native vs Flutter

React Native👍

- 네이티브 앱 운영체제 상에서 가능한 위젯을 사용해야 하는 경우
- 디자인이 iOS 혹은 Android 앱처럼 보이게끔 만들고 싶은 경우

Flutter👍

- 세밀한 디자인 요구사항이 있거나 100% 커스터마이징하고 싶은 경우
- 외부 패키지에 의존하지 않고 고수준의 애니메이션을 구현하고 싶은 경우
